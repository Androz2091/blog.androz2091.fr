### Computer 🖥️

Yep... the core of the question. Before talking about the tools, let's discuss the device they run on.

**Final choice**: [Mac Mini M1](https://www.apple.com/fr/mac-mini/) (desktop, main), [XPS Dell 13](https://www.dell.com/fr-fr/shop/dell-laptops/xps-13/spd/xps-13-9310-laptop) (laptop, secondary) running **Linux**

All the software presented below is necessarily compatible with both MacOS and Linux. I appreciate my Mac Mini M1 for its performance, its very compact size, and MacOS (even though the ideal would be Linux as the main computer).

My laptop is for when I'm on the go or when I absolutely need Linux (like here: [How to clone your cafeteria card?](https://blog.androz2091.fr/copy-nfc-card/)).

### Database Client 🗃️

**Final choice**: [DBeaver](https://dbeaver.io/), a fast, elegant, and very complete cross-platform open-source client supporting PostgreSQL, MySQL, MongoDB, and even cloud databases like AWS.  

**Economic model**: Freemium (enterprise version offered to students)

Considered alternatives:
* [PgAdmin4](https://www.pgadmin.org/) (too bulky and slow, only supports PostgreSQL but entirely open source)
* [Compass](https://www.mongodb.com/products/compass) (lightweight, but only supports MongoDB)
* [DataGrip](https://www.jetbrains.com/fr-fr/datagrip/) (bulky, paid, though very comprehensive)

### Git Client 📦

**Final choice**: [GitKraken](https://www.gitkraken.com/), an elegant and very complete cross-platform Git client.

**Economic model**: Premium (professional version free via GitHub Student)

Considered alternatives:
* [Github Desktop](https://desktop.github.com/) (less comprehensive, not cross-platform)
* [Git CLI](https://git-scm.com/) (not very efficient, if you have a UI, you might as well use it)

### Text Editor 📝

> Mainly used for JS/TS

**Final choice**: [Visual Studio Code](https://code.visualstudio.com/), a rather lightweight, cross-platform open-source text editor, complete thanks to extensions.

**Price**: Free

Considered alternatives:
* [WebStorm](https://www.jetbrains.com/fr-fr/webstorm/) (paid, bulky, though more comprehensive than VS Code)
* [Atom](https://atom.io/) (also pleasant, but project archived...)
* [Sublime Text](https://www.sublimetext.com/) (paid)

### SSH Client 📡

**Final choice**: [Termius](https://termius.com/), a free and open-source cross-platform SSH client.

**Economic model**: Freemium (premium version offered to students)

Considered alternatives:
* [PuTTY](https://www.putty.org/) (bulky, not truly effective cross-platform)
* [SSHception](https://git.kaki87.net/KaKi87/sshception) (still in beta but promising project)

### API testing 📡

**Final choice**: [Postman](https://www.postman.com/), a free and open-source cross-platform API testing client.

**Price**: Free (premium version unnecessary)

Considered alternatives: none for the moment.

### HTTP Proxy 🌐

**Final choice**: [mitmproxy](https://mitmproxy.org/), a free and open-source HTTP proxy.

**Price**: Free

Considered alternatives:
* [Charles](https://www.charlesproxy.com/) (paid, proprietary, bulky)
* [ProxyMan](https://proxyman.io/) (very intuitive interface but proprietary and not cross-platform)

### Web Analytics 📈

**Final choice**: [Umami](https://umami.is/), a free and open-source web analysis tool.

**Price**: Free (self-hosted)

Considered alternatives:
* [Google Analytics](https://analytics.google.com/) (much more comprehensive, but less privacy-friendly, proprietary)

### Accounting 💰

**Final choice**: [Pennylane](https://www.pennylane.fr/), a very comprehensive, pleasant online accounting software with quick support.

**Price**: 28€/month

Considered alternatives:
* [Abby](https://www.abby.fr/) (very promising, but not yet comprehensive enough - does not support invoicing in English and dollars)
* [Fiverr Workspace](https://workspace.fiverr.com/) (lacks key features - no bank connection possible)

### Time Management 🕰️

**Final choice**: [Time Tagger](https://timetagger.app/), an innovative, developer-friendly open-source time management software.

**Price**: Free (self-hosted)

Considered alternatives:
* [Toggl](https://toggl.com/) (paid, proprietary, more cumbersome to configure than Time Tagger)
* [Fiverr Workspace](https://workspace.fiverr.com/) (too few features - lacks Pomodoro)
