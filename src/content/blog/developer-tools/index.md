Certains logiciels m'ont changé la vie, voici ce que j'ai retenu après réflexion et utilisation ;)

### Ordinateur 🖥️

Eh oui... le coeur de la question. Avant de parler des outils, parlons de l'appareil sur ceux-ci vont tourner.

**Choix définitif**: [Mac Mini M1](https://www.apple.com/fr/mac-mini/) (fixe, principal), [XPS Dell 13](https://www.dell.com/fr-fr/shop/dell-laptops/xps-13/spd/xps-13-9310-laptop) (portable, secondaire) sous **Linux**

Tous les logiciels présentés ci-dessous sont forcément compatibles à la fois avec MacOS et Linux. J'apprécie mon Mac Mini M1 pour ses performances, sa taille très réduite et MacOS (même si l'idéal serait un Linux en ordi principal).

Mon ordi portable me sert en déplacement ou lorsque j'ai absolument besoin de Linux (comme ici : [Comment cloner sa carte de cantine ?](https://blog.androz2091.fr/copy-nfc-card/)).

### Client BDD 🗃️

**Choix définitif** : [DBeaver](https://dbeaver.io/), un client cross-platform open source rapide, élégant et très complet supportant PostgreSQL, MySQL, MongoDB, et même les bases cloud type AWS.  

**Modèle éco** : Freemium (version entreprise offerte aux étudiants)

Alternatives considérées :
* [PgAdmin4](https://www.pgadmin.org/) (trop lourd et trop lent, ne supporte que PostgreSQL mais entièrement open source)
* [Compass](https://www.mongodb.com/products/compass) (léger, mais ne supporte que MongoDB)
* [DataGrip](https://www.jetbrains.com/fr-fr/datagrip/) (lourd, payant, même si très complet)

### Client Git 📦

**Choix définitif** : [GitKraken](https://www.gitkraken.com/), un client Git cross-platform élégant et très complet.

**Modèle éco** : Premium (version professionelle gratuite via GitHub Student)

Alternatives considérées :
* [GithubDesktop](https://desktop.github.com/) (moins complet, pas de cross-platform)
* [Git CLI](https://git-scm.com/) (assez peu efficace, si on dispose d'une UI, autant l'utiliser)

### Éditeur de texte 📝

> Utilisé principalement pour du JS/TS

**Choix définitif** : [Visual Studio Code](https://code.visualstudio.com/), un éditeur de texte cross-platform open source plutôt léger, cross-platform et complet grâce aux extensions.

**Prix** : Gratuit

Alternatives considérées :
* [WebStorm](https://www.jetbrains.com/fr-fr/webstorm/) (payant, lourd, même si bien que plus complet que VS Code)
* [Atom](https://atom.io/) (également agréable, mais projet archivé...)
* [Sublime Text](https://www.sublimetext.com/) (payant)

### Client SSH 📡

**Choix définitif** : [Termius](https://termius.com/), un client SSH cross-platform open source et gratuit.

**Modèle éco** : Freemium (version premium offerte aux étudiants)

Alternatives considérées :
* [PuTTY](https://www.putty.org/) (lourd, pas cross-platform vraiment efficace)
* [SSHception](https://git.kaki87.net/KaKi87/sshception) (projet encore en bêta mais prometteur)

### API testing 📡

**Choix définitif** : [Postman](https://www.postman.com/), un client API testing cross-platform open source et gratuit.

**Prix** : Gratuit (version premium inutile)

Alternatives considérées : aucune pour le moment.

### Proxy HTTP 🌐

**Choix définitif** : [mitmproxy](https://mitmproxy.org/), un proxy HTTP open source et gratuit.

**Prix** : Gratuit

Alternatives considérées :
* [Charles](https://www.charlesproxy.com/) (payant, propriétaire, lourd)
* [ProxyMan](https://proxyman.io/) (interface très intuitive mais propriétaire et pas de cross-platform)

### Web Analytics 📈

**Choix définitif** : [Umami](https://umami.is/), un outil d'analyse de site web open source et gratuit.

**Prix** : Gratuit (self-hosted)

Alternatives considérées :
* [Google Analytics](https://analytics.google.com/) (bien plus complet, mais moins respectueux de la vie privée, propriétaire)

### Comptabilité 💰

**Choix définitif** : [Pennylane](https://www.pennylane.fr/), un logiciel de comptabilité en ligne français très complet, très agréable avec un support rapide.

**Prix** : 28€/mois

Alternatives considérées :
* [Abby](https://www.abby.fr/) (très prometteur, mais pas encore assez complet - ne supporte pas la facturation en anglais et dollars)
* [Fiverr Workspace](https://workspace.fiverr.com/) (manque fonctionnalités clefs - pas de connexion bancaire possible)

### Gestion du temps 🕰️

**Choix définitif** : [Time Tagger](https://timetagger.app/), un logiciel de gestion du temps open source, innovant et developer-friendly.

**Prix** : Gratuit (self-hosted)

Alternatives considérées :
* [Toggl](https://toggl.com/) (payant, propriétaire, plus lourd à configurer que Time Tagger)
* [Fiverr Workspace](https://workspace.fiverr.com/) (trop peu de fonctionnalités - manque Pomodoro)
